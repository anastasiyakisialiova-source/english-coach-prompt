# English Coach — Daily Prompt

Этот публичный repo содержит один файл `today.txt` с промптом дня для голосовой сессии.

Обновляется автоматически каждое утро через morning workflow в основном (приватном) репо.

Используется iOS Shortcut "English Coach Today" — он качает `today.txt`, кладёт в clipboard и открывает Custom GPT.

Raw URL: https://raw.githubusercontent.com/anastasiyakisialiova-source/english-coach-prompt/main/today.txt
